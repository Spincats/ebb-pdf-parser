# Web Electronic Bluebook PDF Parser

## Why?

This is a quick project to automate the tedious task of copying student multiple choice answers out of the EBB PDFs and putting them into a spreadsheet. Because I could, I went ahead and put these into the grading spreadsheet template that I've developed. But feel free to copy the answers out into whatever template you prefer.

## Is this safe?

You do need to trust me ever so slightly not to be completely lying to you. As long as you can do that, then I can assure you that this is totally safe. This is static code, delivered via Github, hosted on Github Pages, and all processing of the student data is handled completely in your browser on your local machine. There is no database to compromise, there are no records kept anywhere.

## My files don't process correctly

I don't have an official exam PDF specification and am in no way affiliated with EBB. This code works on the exams that I've seen as examples over the course of a few years, but it might not work on every exam and might not work if EBB changes the format moving forward. Sorry. If you have a specific exam that doesn't work or if EBB has updated the format, feel free to reach out and let me know.
